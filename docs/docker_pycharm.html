Connect Docker to a PyCharm Project

First, create a Python repository in GitHub. Make sure to add a README and .gitignore file. Next, copy the repo URL because you will need it to clone the repository through your CLI shell.

In the shell, go to the directory where you would like to save your project. Enter the command “git clone [REPOSITORY URL]” using your Python repository URL. To go to the newly made directory, enter the command “cd [DIRECTORY NAME]/”. The directory name should be the same as the name of your repository. Use the command “git status ” to see if your project is up to date.
