Containerization

Containerization serves as an alternative form to virtualization. All applications are encapsulated in isolated spaces of code called containers. A container is essentially a portable computing environment that runs on top of the OS. By packaging up the software, sharing infrastructure and cloud services becomes quick and efficient. The portability aspect plays a great deal in its ability to run reliably on other machines.
